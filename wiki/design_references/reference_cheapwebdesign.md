# Design Reference — cheapwebdesign.co.uk

The look Web Lift's landing page replicates (Doug's explicit instruction). Studied live 2026-05-29.

Screenshots (this folder): `cheapwebdesign-home.png` (full homepage), `cwd-hero.png`, `cwd-pricing.png`, `cwd-contact.png`.

## What makes the look work
- **Big, bold, friendly.** Huge black display headline (Outfit, 800 weight), tight leading. Confident, not corporate.
- **Hand-drawn blue marker scribble** circling/underlining one key word in headings — the signature motif.
- **Fully-rounded pill buttons.** Royal blue primary, sky-blue secondary, black "Get Quote".
- **Big rounded cards** (~22px radius): pricing uses one filled royal-blue card + one white card, with a giant price number and star-emoji feature bullets.
- **Trust everywhere:** Google 4.9★ row, floating circular customer-avatar photos around the hero.
- **Maroon announcement bar** up top with a running offer + rating + phone.
- **Cheerful emoji** in feature lists (🚀🎨📱). Warm, human, anti-jargon.

## Sampled tokens
- Royal `#2F3FF7` · Sky `#22B5FC` · Black `#0A0A0A` · Maroon `#9C1F06` · BG `#F3F5F9` · Gold `#FBB014`
- Font: Outfit (Google Fonts)

## How Web Lift diverges
- Web-Lift-specific copy (local-SMB redesigns, not generic web design).
- Added a **before/after drag slider** in the hero (reference has none) — demonstrates the value prop in one gesture.
- Two tiers: Website Redesign from £750, Brand-new Site from £950.
