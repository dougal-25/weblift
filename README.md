# Web Lift

Done-for-you website redesigns for local small businesses. One flat fee from £750 — no jargon, no hassle. Domain: web-lift.co.uk.

## Quick start

```bash
cd projects/weblift
python3 -m http.server 8753
# open http://localhost:8753/index.html
```
